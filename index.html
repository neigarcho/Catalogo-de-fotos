JavaScript 1995

JavaScript

El lenguaje de la web. Con Node.js conquistó el backend; con React y Vue, el frontend moderno.

Rust 2010

[ 003 ]

Rust

Seguridad de memoria sin garbage collector. El lenguaje más amado por los desarrolladores año tras año.

C 1972

[ 004 ]

C

El abuelo de los lenguajes modernos. Sistemas operativos, kernels y embebidos siguen dependiendo de él.

TypeScript 2012

[ 005 ]

TypeScript

JavaScript con superpoderes. Tipado estático que detecta errores antes de que tu código llegue a producción.

Go 2009

[ 006 ]

Go

Concurrencia nativa, compilación ultrarrápida y sintaxis minimalista. La apuesta de Google para el backend.

SQL 1974

[ 007 ]

SQL

El lenguaje de las bases de datos relacionales. Más de 50 años después, sigue siendo insustituible.

Swift 2014

[ 008 ]

Swift

El lenguaje de Apple. Rápido, seguro y expresivo para construir apps iOS y macOS de clase mundial.

Java 1995

[ 009 ]

Java

Write once, run anywhere. El pilar del desarrollo empresarial, Android y sistemas de gran escala.
